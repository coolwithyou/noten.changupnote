# 창업노트 문서 허브

이 폴더는 창업노트 시스템을 **읽는 사람에 맞춰** 정리한 문서 모음입니다.
프로젝트 루트에는 설정·빌드·에이전트 지시 파일만 두고, 기획/설계/리서치/IR/프로토타입 산출물은
이 `docs/` 폴더 아래 카테고리별로 둡니다.

## 문서 작성 규칙

1. **독자를 먼저 정한다.** 비개발자용이면 코드·전문용어를 최소화하고 비유·예시·표로 설명한다.
2. **코드가 정답이다.** 해설 문서는 코드 동작을 풀어쓴 것이므로, 로직이 바뀌면 해당 문서도 갱신한다. 각 문서 하단에 "근거 코드 위치"를 남긴다.
3. **카테고리 README의 목록에 등록한다.** 새 문서를 추가하면 해당 폴더 `README.md`의 표에 한 줄 추가한다.
4. **파일명은 한국어 슬러그**로 읽기 쉽게 (`핵심-매칭-로직-쉽게-이해하기.md`).
