# 창업노트 공고 검수 가이드

안녕하세요. 창업노트 공고 검수에 함께해 주셔서 감사합니다.

이 문서는 처음 검수하실 때 필요한 작업 순서와 판정 기준을 간단히 정리한 안내서입니다. 검수 화면은 아래 주소에서 이용하실 수 있습니다.

- 검수 화면: https://ops.changupnote.com/review
- 로그인 정보: 운영 담당자가 보내드린 개인 안내 문서에서 확인해 주세요.

## 1. 로그인해 주세요

1. 검수 화면을 열고 **Google로 계속**을 선택해 주세요.
2. 안내받으신 `@noten.im` 계정으로 로그인해 주세요.
3. Google 로그인이 어려운 경우에는 개인 안내 문서의 이메일과 임시 비밀번호를 이용해 주세요.
4. 로그인 후 `검수 배정 큐`가 보이면 정상적으로 접속하신 것입니다.

개인 안내 문서에는 로그인 정보가 포함되어 있습니다. 다른 분에게 전달하거나 공용 메신저에 올리지 마시고, 로그인을 확인하신 뒤에는 안전하게 보관해 주세요.

## 2. 무엇을 확인하나요?

AI가 공고에서 정리한 지원 조건이 실제 공고 내용과 일치하는지 확인해 주시면 됩니다. 문장의 표현보다는 조건의 의미가 정확한지 살펴봐 주세요.

> **화면에 표시된 조건과 값이 공고 원문에서도 같은 뜻으로 안내되고 있나요?**

원문만으로 확정하기 어려운 내용은 추측하지 않으셔도 됩니다. 이 경우 `판단 불가`를 선택하시고 확인이 어려웠던 이유를 간단히 남겨 주세요.

## 3. 기본 검수 순서

1. `검수 배정 큐`에서 검수할 공고의 `검수 시작`을 눌러 주세요.
2. 버튼에 `상세 불러오는 중`이 표시되면 화면이 열릴 때까지 잠시 기다려 주세요.
3. 가운데의 `현재 검수 질문`을 먼저 읽어 주세요.
4. 왼쪽에 기본으로 열린 `AI 분석 문서`에서 전체 맥락을 확인해 주세요.
5. `저장된 공고 원문`, `공고 원문 페이지`, HWP/HWPX 첨부 문서를 함께 살펴보며 실제 조건과 비교해 주세요.
6. 가운데의 `검수자 입력` 영역에서 판정을 선택하고, 필요한 경우 판정 사유를 작성해 주세요.
7. `저장하고 다음 미판정 항목`을 누르면 저장 후 다음 항목이 자동으로 열립니다.
8. 마지막 항목까지 마치면 `저장하고 공고 검수 완료`를 눌러 주세요.
9. 오른쪽에 완료 안내가 나타나면 `공고 목록으로 돌아가기`를 눌러 다음 공고를 진행해 주세요.

선택하신 판정은 진한 배경으로 표시됩니다. 오른쪽 `전체 필드`에서 저장 완료, 저장 전 변경, 미판정 상태를 확인하실 수 있습니다.

체크 아이콘과 판정명이 표시된 항목은 이미 저장된 항목입니다. 다시 확인하실 필요는 없으며, 수정이 필요한 경우에만 해당 항목을 열어 변경 후 저장해 주세요.

목록 상단과 상세 화면 오른쪽의 진행률은 숫자와 막대로 함께 표시됩니다. 판정을 선택한 뒤 저장까지 완료하면 진행률에 반영됩니다.

## 4. 화면은 이렇게 구성되어 있습니다

- **왼쪽 — 판정 근거:** AI 분석 문서, 저장된 공고 원문, 공고 원문 페이지, 첨부 문서
- **가운데 — 현재 검수 질문:** 확인할 조건과 값, 원문 근거, 판정 선택, 판정 사유
- **오른쪽 — 전체 필드:** 공고에 포함된 검수 항목과 각 항목의 진행 상태

`기술 정보 보기`는 일반적인 검수에는 사용하지 않으셔도 됩니다. 운영 담당자가 별도로 요청한 경우에만 확인해 주세요.

## 5. 판정은 이렇게 선택해 주세요

| 판정 | 선택 기준 | 판정 사유 |
|---|---|---|
| **정확** | 화면의 조건과 값이 공고 원문과 같은 의미입니다. | 선택 |
| **수정 필요** | 조건은 원문에 있지만 값, 범위, 조건의 성격 등이 다릅니다. | 필수 |
| **오류** | 화면에 표시된 조건을 공고 원문에서 찾을 수 없거나 다른 내용을 잘못 해석했습니다. | 필수 |
| **판단 불가** | 제공된 원문과 첨부 문서만으로는 확정하기 어렵습니다. | 필수 |

`수정 필요`를 선택하실 때에는 아래 두 가지를 함께 적어 주시면 좋습니다.

- 어떤 부분이 다른지
- 원문 기준으로 어떤 내용이 맞는지

예시: `원문에서는 39세 이하만 확인되며 19세 이상이라는 하한 조건은 찾기 어렵습니다. 하한 조건을 제외해야 합니다.`

## 6. 빠진 조건이 있는지 묻는 항목

일부 항목에서는 해당 주제의 조건이 공고에 정말 없는지 확인합니다.

| 판정 | 선택 기준 |
|---|---|
| **없음 확인** | 공고 원문과 첨부 문서에서 해당 조건을 찾을 수 없습니다. |
| **누락 있음** | 공고에 해당 조건이 있지만 분석 내용에는 빠져 있습니다. |

`누락 있음`을 선택하실 때에는 빠진 원문 내용과 해당 조건이 필수, 우대, 제외 중 어디에 해당하는지 적어 주세요.

## 7. 확인하실 때 특히 주의해 주세요

1. **숫자와 기간** — 나이, 업력, 매출, 인원, 기간이 원문과 같은지 확인해 주세요.
2. **범위 표현** — `이상`, `이하`, `초과`, `미만`의 방향이 바뀌지 않았는지 확인해 주세요.
3. **조건의 성격** — 필수 조건, 우대 조건, 제외 조건이 서로 바뀌지 않았는지 확인해 주세요.
4. **목록 누락** — 지역이나 업종 등 여러 값 가운데 일부가 빠지지 않았는지 확인해 주세요.
5. **중복 항목** — 같은 내용이 반복되어 있다면 하나만 `정확`으로 두고, 나머지는 중복된 내용임을 사유에 적어 `수정 필요`로 선택해 주세요.

`서류 미비 시 제외`, `허위 기재 시 제재`처럼 신청 절차에 관한 안내가 지원 자격으로 표시된 경우에는 `수정 필요`를 선택하고, 절차 안내라는 점을 사유에 적어 주세요.

## 8. 완료한 공고를 다시 확인하려면

1. 배정 큐에서 `완료` 탭을 눌러 주세요.
2. 다시 확인할 공고의 `결과 보기·수정`을 눌러 주세요.
3. 오른쪽 `전체 필드`에서 저장된 판정을 확인해 주세요.
4. 수정이 필요한 항목만 열어 판정이나 사유를 변경해 주세요.
5. `저장 전 변경` 상태를 확인한 뒤 저장 버튼을 눌러 주세요.
6. 수정이 끝나면 `공고 목록으로 돌아가기`를 눌러 주세요.

완료된 공고를 열었다고 모든 항목을 다시 검수하실 필요는 없습니다. 내용을 변경하지 않은 경우에는 추가로 저장되지 않습니다.

## 9. `독립 표본` 항목

`독립 표본` 표시가 있는 항목은 다른 판단을 참고하지 않고 원문만으로 확인해 주시면 됩니다. 다른 분의 답변이 보이지 않는 것이 정상입니다.

판정 결과를 별도로 비교하거나 조정하실 필요는 없습니다. 필요한 후속 확인은 운영 담당자가 진행합니다.

## 10. 첫 번째 공고는 함께 기준을 맞춰 주세요

처음 맡으신 공고는 운영 담당자와 함께 판정 기준을 맞추는 용도로 진행합니다.

1. 먼저 원문을 기준으로 판정해 주세요.
2. 저장하기 전에 운영 담당자와 판정 및 사유를 함께 확인해 주세요.
3. 기준이 다른 부분을 정리한 뒤 저장해 주세요.
4. 이후 공고부터는 같은 기준으로 이어서 진행해 주세요.

## 11. 첨부 문서 확인 방법

1. 왼쪽 `공고 첨부 문서`에서 파일명을 누르면 새 탭에서 미리보기가 열립니다.
2. `HWPX 비표준 감지` 안내가 나타나면 `자동 보정 (권장)`을 눌러 주세요.
3. 글꼴 관련 안내가 나타나면 `대체 글꼴로 보기`를 선택하셔도 됩니다.
4. 문서를 모두 불러오면 필요한 내용을 확인한 뒤 원래 검수 탭으로 돌아와 주세요.
5. 미리보기에서 내용을 확인하는 동안 원래 검수 화면의 입력 내용은 그대로 유지됩니다.

미리보기가 열리지 않으면 공고 원문 페이지의 첨부 링크를 확인해 주세요. 그래도 내용을 확인할 수 없다면 추측하지 마시고 `판단 불가`를 선택한 뒤 운영 담당자에게 알려 주세요.

## 12. 도움이 필요하실 때

문제가 발생하면 운영 담당자에게 아래 내용을 보내 주세요.

- 문제가 발생한 시각
- 확인 중이던 공고 제목 또는 화면 주소
- 화면에 표시된 오류 메시지
- 새로고침 후에도 같은 문제가 나타나는지

배정 목록이 비어 있거나 판정 기준이 애매한 경우에도 편하게 운영 담당자에게 문의해 주세요.
