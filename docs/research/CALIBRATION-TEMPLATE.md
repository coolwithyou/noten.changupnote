# 외부 대조(External Calibration) 실행 템플릿

마스터 설계 17장 "관문 공통 의례"의 실행 절차. 새 세션의 Claude는 관문 착수 전 이 템플릿대로 수행한다.

## 절차 (5단계)

1. **전제 목록화** (메인/감독자): 이번 관문이 딛고 있는 설계 결정을 마스터 문서에서 뽑아 3~7개로 목록화한다. 예: Gate 2라면 "bbox는 layout 엔진 소유", "reconciliation 신뢰도 규칙", "layout 엔진 후보군"
2. **리서치 위임** (Opus 에이전트 2개 병렬, 아래 프롬프트 골격 사용): 기술 축 + 제품·규제 축. 직전 대조 이후 기간의 최신 정보를 요구한다
3. **산출 검수** (감독자): 출처 실재 확인 표본 검사, 벤더 주장 vs 실측 구분 여부, 불확실성 명시 여부
4. **판정** (감독자): 전제별 유지/보강/재고 판정표 작성. "측정 전 채택 금지" — 새 도구/기법은 golden set 측정 후보로만 등재
5. **반영**: 마스터 문서에 결정 기록(해당 장 + 1장 관련 문서 이력), 핸드오프 문서에 후속 태스크 등재, 커밋
