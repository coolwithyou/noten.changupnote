# 창업노트 과금 모델 — 쉬운 결정 페이퍼

작성일: 2026-07-14  
상태: 유료 베타를 위한 가격 초안

## 한 줄 결론

> **공고를 찾고 확인하는 일은 무료로 제공하고, 공고를 놓치지 않게 챙겨주고 신청서를 작성해 주는 일에 돈을 받는다.**

고객에게는 복잡한 크레딧을 보여주지 않는다. 고객은 **무료**, **월 9,900원 멤버십**, **대행사·기관 문의**만 보면 된다.

## 상품은 세 가지다

| | 무료 | 멤버십 | 대행사·기관 |
|---|---:|---:|---:|
| 가격 | 0원 | **월 9,900원** | 문의 |
| 공고 찾기와 지원 가능 여부 확인 | 전체 이용 | 전체 이용 | 계약에 따라 제공 |
| 새 공고 알림 | 주 1회 이메일 | **실시간 맞춤 알림** | 계약에 따라 제공 |
| 마감 알림 | 없음 | 제공 | 계약에 따라 제공 |
| 애매한 조건 확인 질문 | 월 3회 | **무제한** | 계약에 따라 제공 |
| 신청서 도우미 | 가입 시 2회 | **매달 15회** | 계약에 따라 제공 |
| 관리할 수 있는 회사 | 1개 | 1개 | 여러 개 |

신청서 도우미가 더 필요하면 **5회 이용권을 3,900원**에 살 수 있다.

신청서 도우미 1회에는 다음 작업이 모두 들어간다.

- 신청서 서식 분석
- 신청서 초안 작성
- 신청서 자동 채움

멤버십에서 받은 횟수는 지급일로부터 **60일**, 가입 축하 2회는 **90일** 동안 사용할 수 있다.

## 왜 이 구조인가

무료 기능은 고객을 모으는 입구다. 유료 기능은 고객의 시간과 노동을 줄여준다.

- 공고 정보와 판정은 무료로 열어 경쟁력을 만든다.
- 실시간 알림은 고객이 좋은 공고를 놓치지 않게 한다.
- 신청서 도우미는 가장 번거로운 실제 작업을 대신한다.
- 여러 회사를 관리하는 대행사는 지불 의사가 높으므로 별도로 계약한다.

요금제를 여러 개 만들지 않는 이유도 단순하다. 지금은 고객이 어느 요금제를 고르는지보다 **월 9,900원을 낼 만큼 가치를 느끼는지** 먼저 확인해야 한다.

## 가격은 성립하는가

| 항목 | 금액 |
|---|---:|
| 신청서 도우미 1회 예상 원가 | 약 330원 (환율 1,500원 기준) |
| 멤버십의 회당 가격 | 660원 (9,900원 ÷ 15회) |
| 추가 이용권의 회당 가격 | 780원 (3,900원 ÷ 5회) |

멤버십 15회를 모두 사용해도 예상 공헌이익률은 약 **42%**다. 절반 정도 사용하면 약 **67%**다.

추가 이용권은 멤버십보다 회당 약 18% 비싸다. 가끔 더 필요한 고객은 이용권을 사고, 자주 쓰는 고객은 멤버십을 유지하게 만드는 가격이다.

환율이 약 **1,780원**을 넘지 않는 한 이 가격은 그대로 유지할 수 있다. 넘으면 내부 크레딧 기준만 조정하면 된다 (가격 인상은 7일 전 고지).

## 고객에게는 이렇게 설명한다

가격 페이지의 첫 문장은 이것으로 충분하다.

> 공고를 찾고 확인하는 건 전부 무료예요.  
> 월 9,900원이면 새 공고를 놓치지 않게 챙겨드리고, 신청서 작성을 매달 15회 도와드려요.

가격 페이지에는 카드 두 개만 둔다.

1. **무료로 시작하기**
2. **멤버십 월 9,900원**

그 아래에는 세 가지만 안내한다.

- 무료는 공고를 **찾고 확인하는 서비스**다.
- 멤버십은 공고를 **놓치지 않고 신청까지 끝내는 서비스**다.
- 여러 회사를 관리하는 대행사·기관은 별도로 문의한다.

## 내부에서는 기존 크레딧을 유지한다

고객은 횟수만 보지만, 결제와 환불을 위해 내부 크레딧 시스템은 그대로 사용한다.

| 고객이 보는 값 | 내부 값 |
|---|---:|
| 신청서 도우미 1회 | 780크레딧 |
| 가입 축하 2회 | 1,560크레딧 |
| 멤버십 월 15회 | 11,700크레딧 |
| 추가 이용권 5회 | 3,900크레딧 |

반드시 지킬 원칙은 네 가지다.

1. 신청서 도우미 1회는 항상 **780크레딧으로 고정**한다.
2. 서식 분석, 초안 작성, 자동 채움을 따로 차감하지 않고 **한 번으로 묶는다**.
3. 고객 화면과 안내 문구에는 크레딧, 환율, 토큰을 표시하지 않는다.
4. 잔액, 사용 내역, 결제, 환불도 모두 **회 단위**로 설명한다.

따라서 가격 페이지, 잔액, 충전, 사용 내역, 횟수 소진 안내, 약관과 환불 정책을 회 단위 표현으로 바꿔야 한다. 내부 원장, 결제, 환불, 만료 처리, 감사 로그는 기존 시스템을 재사용한다.

## 유료 베타에서 확인할 것

초기 100개 기업에는 정가를 바꾸지 않고 **월 4,950원, 12개월 고정** 얼리버드 할인을 제안한다.

| 확인할 신호 | 1차 목표 |
|---|---:|
| 무료 고객의 유료 전환율 | 3% 이상 |
| 가입 3개월 뒤 유지율 | 60% 이상 |
| 유료 고객의 월 1회 이상 사용률 | 50% 이상 |

추가 이용권을 반복해서 사는 고객이 많아지면 상위 요금제를 만든다. 여러 회사 관리 문의가 쌓이면 대행사·기관용 계약 상품을 구체화한다.

## 아직 결정할 것

1. 유료 플랜 이름을 “멤버십”으로 확정할지
2. 비수기에 구독을 잠시 멈추는 기능을 제공할지
3. NICE·CODEF 기업정보 정밀 갱신의 가격을 얼마로 할지
4. 무료 확인 질문 월 3회가 적절한지
5. 원가 계산의 모델과 토큰 사용량 가정을 승인할지 (환율은 1,500원으로 확정 — 2026-07-14)

## 최종 판단

출시 시점에는 무료와 월 9,900원 멤버십만 보여준다. 실제 사용 데이터에서 더 많이 쓰는 고객이 확인된 뒤 상위 요금제를 추가한다.

> **지금은 선택지를 늘릴 때가 아니라, 고객이 돈을 내는 이유를 하나로 선명하게 만들 때다.**
