> 배경: 기업 데이터 소싱 클러스터(CODEF·팝빌·NICE·Apick·KCOMWEL·SMPP)를 유료 벤더 1곳으로
> 통합하기 위한 3사 병렬 견적. 비교 기준은 `2026-07-13` 세션의 커버리지 표 참조.

## 공통 배경 (3사 공통으로 문의문에 포함됨)

- 서비스: 창업노트(changupnote.com) — 사업자번호 기반으로 공공 지원사업을 매칭하고 지원서 작성을 돕는 SaaS
- 이용 목적: **이용자 본인 기업 정보 조회 대행** (제3자 기업 정보의 재제공·재판매 아님 — 계약서 명시 요청)
- 조회 방식: 이용자가 본인 명의 간편인증으로 동의한 뒤 자사 서버가 API 호출
- 물량 시나리오: 월 1,000건 / 10,000건 / 50,000건 3구간 견적 요청

## 공통 질문 체크리스트 (3사 동일)

1. 상품별 건당 단가 (3구간 시나리오별) + 볼륨 할인 구간
2. 월 기본료·최소 약정·초기 셋업비 유무
3. 과금 기준: 성공 응답 건만 과금인지 / 간편인증 미완료·중도이탈·빈 응답(무신고 등) 과금 여부
4. 조회 결과의 자사 DB 저장·캐싱 허용 범위와 보존 기간 제한(TTL) — 재조회 과금 최소화 목적
5. 계약 절차·심사 소요 기간·필요 서류, 초기 스타트업 계약 가능 여부
6. 개인정보 처리 위수탁 관계 정리(계약서상 지위), 재제공 제한 조항의 정확한 범위
7. 간편인증 지원 수단(카카오·네이버·PASS·금융인증서 등)과 인증 세션 재사용(SSO) 정책
8. 테스트/개발 환경 제공 여부, 응답 필드 명세서 제공, SLA·정기점검 시간·장애 공지 채널

**CODEF 개별 질문**
