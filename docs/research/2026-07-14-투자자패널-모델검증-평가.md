# 투자자 패널 모델 검증 — 2라운드 평가 기록

작성일: 2026-07-14
방법: 랜딩 카피 배치안·신뢰지표·IR 기술 슬라이드 3종을 **문서를 처음 보는 AI 투자자 페르소나**에게 읽혀 평가. 1차(4인)는 문서만으로, 2차(3인)는 창업자 구두 보충(아래 교정 가정)을 더해 재평가. 페르소나별 독립 평가 후 종합.

## 0. 교정 가정 (2차 평가의 전제 — 창업자 확인 사항)

1. **단계**: 예비창업(법인 설립 전), 시드 유치 목적. 사용자·매출 부재는 이 단계의 정상 상태.
2. **수익모델**: '똑닥'식 심플·저가 구독 — ①공모 알리미/리마인더 ②지원서 신청 도우미(AI). "기업이라면 일단 구독해두는" 포지션.
3. **페인**: 신청서 작성에 애먹는 신청자 다수 + 지원 기관의 CS 인력 부족(양면).
4. **확장**: 사업체가 모이면 ③지원사업 광고 ④커스터마이징 공고 도우미/공모 대행.

## 1. 판정 변화 (1차 → 2차)

| | 1차 (문서만, 운영 서비스로 오인) | 2차 (예비창업 시드 + 수익모델 보충) |
|---|---|---|
| 시드 VC | 미팅만 — "수요 지표 0" | **조건부로 쓴다** |
| 테크 DD | 미팅만 — "해자는 전부 미래형" | (1차만 참여) |
| 구독 투자자 | 미팅만 — "숫자 100% 빌드 지표" | **쓴다 (마일스톤 트랜치 조건)** |
| 도메인 회의론자 | 미팅만 — "기계의 자서전" | **세 가지 채워오면 쓴다** |

교훈: 1차의 혹평 상당수는 **문서가 단계·수익모델을 안 담아서 생긴 오해**였다. 실제 투자자도 같은 자료로 같은 오해를 한다 → 데크에 단계 선언과 수익모델 장 필수.

## 2. 2차 평가 종합

### 2.1 일치된 강점 (시드 기준)

- **빌드 증거 상위 1%**: 법인 설립 전에 공고 31,408건·조건 87,840건·판정 147,159건 E2E + 운영급 신뢰 장치(재계산 무실패, 버전 귀속). "'만들 수 있는가'는 이미 답이 나왔다."
- **지적 정직성**: 금지 표현 목록·적대 검증·자기 기각 — "창업자의 자기기만 리스크가 구조적으로 낮다", "실사 비용을 크게 낮춘다."
- **원가 구조**: O(공고) 상각(공고당 ≈$0.003)·판정 한계비용 0 → 저가 구독의 마진이 태생적으로 방어됨.
- **문제 선택의 눈**: 접수처 225종 실측으로 자동 제출을 스스로 기각, HWP 60/60 — "돈을 태우기 전에 재는 창업자."
- **결격 3축**: 확인되면 무료 대안(보조금24) 대비 실질 차별점 후보.

### 2.2 모델에 대한 공통 지적

1. **똑닥 유추는 빈도에서 깨진다** (3인 전원). 똑닥 = 고빈도 니즈 + 공급측(병원) 병목 장악 후 과금, 그마저 유료화 고전. 지원사업은 기업당 연 1~3회의 시즌성 행위 → 알리미 구독은 대기성 상품이고, 해지 곡선은 4~12개월 차가 고비. **"월간 사용 이벤트가 무엇인가"** 에 답해야 한다.
2. **구두 피칭과 코드의 불일치**: 피칭은 "심플 정액 구독", 저장소는 크레딧 종량(포트원) + 크레딧 플랜 + 별개 billing_* 공존 — "과금 전략 미수렴" 신호로 읽힘. → 해소안은 [[2026-07-14-과금모델-초안과-프라이싱-구성]].
3. **④공모 대행의 이중 문제**: (a) 자체 실측·설계 원칙("자동 대리제출 불가", "우리는 대행사가 아니다")과 정면 모순 (b) 유상 서류작성·제출대행은 행정사법(행정사 독점)·중기부 브로커 근절 정책의 경계 — 저장소에 규제 검토 문서 0건(grep 실측). **핵심 재정의 제안(구독 투자자)**: ④를 신청자 대리가 아니라 **모집 기관이 접수 인프라로 채택하는 B2G**로 재배열 — "구독은 수요·데이터 수집 장치, 본 모델은 기관측 인프라"가 되면 똑닥의 '공급측 통합' 수순과 동형이 되고 규제 문제도 소거.
4. **③광고의 광고주 실체**: 공공기관은 사기업 플랫폼 광고비 집행이 구조상 어려움 → 실제 광고주 가설(운영사·AC·금융권 등) 규명 필요.
5. **기관 CS 페인이 주장 상태**: 기관 인터뷰·LOI 등 근거 부재.

### 2.3 수표 조건 (3인 공통 — 텀싯 전)

1. **지불의사 실증** — 고객 인터뷰 30건+ / 대기명단 / 프리세일즈. 기관측 페인도 인터뷰·LOI로.
2. **정확도 게이트 계획** — 사람 검수 골든셋 P/R의 시점·합격선. 특히 거짓 탈락(될 사업을 안 된다고 판정) 허용 한도와 발생 시 대응.
3. **④대행의 법적 구조** — 행정사법 자문, "도구 제공 vs 대행" 경계 (또는 B2G 재정의).

### 2.4 데크 필수 목록 (3인 합집합, 현재 자료에 없는 것)

1. 팀/창업자 — why me, 도메인 접점, 팀 빌딩 계획 ("시드는 결국 사람에 거는 돈")
2. 시장 산수 — SAM(지원사업 활성 탐색층), 단가×전환율 시나리오, 목표 MRR 경로. 기존 브로커·컨설팅 시장 규모 = 지불의사의 간접 증거
3. 요금표 확정안 — 구독 vs 크레딧 관계 정리 (과금모델 문서 참조)
4. 수요 검증 — 인터뷰 n·핵심 인용·대기자, 없으면 시드 자금으로 수행할 검증 설계
5. GTM 채널 — 첫 1,000 유료사 경로와 CAC (세무사·기장, 창업지원기관 제휴, SEO)
6. 경쟁 지도 — 무료 보조금24 대비 "30초 안에 체감되는 유료 가치" 한 장. 결격 3축 조사 방법론 공개
7. 규제 검토 — 행정사법·브로커 근절 정책, CODEF/NICE 개인정보 컴플라이언스
8. 정확도 로드맵 — P/R 게이트 일정·합격선·실패 시 대응
9. 확장 트리거 — ①②→③④ 전환 조건, ③ 광고주 규명, ④ 법적 구조
10. 자금 사용처·마일스톤 — 18개월 내 증명할 것(유료 구독 n사, 기관 파일럿 1곳 등)
11. 데이터 의존 리스크 — 공공 API 정책 변경·보조금24 고도화 시나리오 대응

### 2.5 예상 질문 (미팅 리허설용, 압축)

1. 월 얼마이고, 그 가격을 내겠다는 증거가 몇 건인가? 연 1~3회 신청하는 기업이 12개월 구독을 유지할 사용 루프는? (똑닥도 유료화에서 고전했는데 왜 이 카테고리는 다른가)
2. 무료 보조금24를 두고 30초 안에 체감되는 유료 가치는 무엇인가? 결격 3축을 경쟁사가 6개월 안에 따라오면 무엇이 남는가?
3. 골든셋 P/R은 언제·어떤 합격선으로 통과하나? 거짓 부적격 1건의 신뢰 붕괴는 어떻게 탐지·보상하나?
4. ④대행의 실체는 신청자 대리인가 기관측 B2G 인프라인가? 전자면 행정사법 경계를 어떻게 넘고, 후자면 첫 기관 파일럿·조달 경로는? ③광고의 광고주는 구체적으로 누구인가?
5. 팀은 누구인가? 1인+AI 체제라면 키맨 리스크와 영업·도메인 공백을 시드 자금으로 어떻게 메우나?
6. K-Startup·기업마당이 접근을 조이거나 정부가 매칭을 직접 고도화하면 해자는 무엇인가?

## 3. 한계

- 패널은 AI 페르소나 시뮬레이션 — 실제 투자 심사의 대체가 아니라 리허설·공백 탐지 도구.
- 2차 패널의 "행정사법·브로커 근절" 지적은 방향성 경고로 받되, **실제 법률 자문으로 검증 필요** (패널이 법률 전문가는 아님).
- 1차 패널이 참조한 recommendable 0.61% 등 내부 평가 수치는 2026-07-13 시점 값.
